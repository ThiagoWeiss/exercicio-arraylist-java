# Exercício de ArrayList em Java

Este repositório contém um exercício realizado durante o curso da Alura sobre coleções em Java.

## Objetivo do exercício
- Criar uma lista de produtos usando `ArrayList`.
- Adicionar produtos perecíveis e não perecíveis.
- Exibir os produtos formatados com nome, quantidade, preço e validade (quando aplicável).

## Arquivos
- `Produto.java`: classe base com atributos comuns.
- `ProdutoPerecivel.java`: herda de Produto e adiciona o atributo `validade`.
- `Retorna.java`: classe principal que cria e manipula a lista de produtos.
